# Third-Party Notices

## cJSON

This project links to the external cJSON library and includes a small set of cJSON 1.7-compatible public declarations in `third_party/cJSON.h` so source files have a stable include path.

cJSON is licensed under the MIT License. Upstream project: `DaveGamble/cJSON`.

The cJSON implementation itself is not copied into this source archive. Install the cJSON development/runtime package for the target operating system.
